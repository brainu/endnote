# 不同文献类型的图片附件

在图片栏目可插入图片或者文件，根据文献类型，该栏目可能是图片或者自定义。EndNote 默认所有文献类型都包含图片和说明栏目，因此一个期刊的参考文献可能包含主要参考目录信息，还可能包含插图。

Figure, Chart or Table, Equation 类文献的图片附件相对较特殊，可用于分类图片和文件，其他文献信息可能很少

**注：**Chart or Table 类文献中的图片可 Word 中插入图片时是作为表格插入

增加和删除图片和说明栏目，可依次打开菜单 Edit &gt;&gt; Preferences &gt;&gt; Reference Types，详见：[自定义文献类型](../14RefTypes/Customizing_the_RefTypes.htm)
