# 导出和打印术语表

术语表可以导出和导入，详见[导入术语到术语表](Importing_TermsIntoTrmLsts.htm)

同样还可以导出术语表然后利用文字处理软件打印

**导出术语表**

1. 依次打开菜单 Tools &gt;&gt; Define Term Lists
2. 选择一个需要导出的术语表
3. 选择 Export List
4. 选择位置和名字
5. 保存

术语表实际上是一个文本文件，可以使用记事本打开并打印，当然也可以复制到 Word 中进行排版后再打印
